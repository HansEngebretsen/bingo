## About
Simple bingo game haloween themed, mostly for kids - with options for icons, text or both

## How to Play

1.  On github pages: [https://haaans.com/bingo/](https://haaans.com/bingo/).
2.  Default mode loads the card
3.  As the items are called out (or as you spot them in your favorite Halloween movie!)  [https://haaans.com/bingo/admin](https://haaans.com/bingo/admin) shows the caller card (or click on card + admin mode.
4.  Get five in a row (horizontally, vertically, or diagonally) to win!

## Fun Details
-   **Easter Eggs:** Keep an eye out for some spooky surprises and animations!
-   **Customizable:** You can toggle between different display modes for the bingo card.

## Local Development

Want to run the project locally and maybe add your own spooky touches?

1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Run the development server:**
    ```bash
    npm run dev
    ```
