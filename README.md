Welcome to Spooky Bingo, a fun, interactive bingo game with a delightful Halloween theme!

## How to Play

1.  Open the site at [https://haaans.com/bingo/](https://haaans.com/bingo/).
2.  You'll be presented with a bingo card filled with spooky-themed items.
3.  As the items are called out (or as you spot them in your favorite Halloween movie!), click on the corresponding square on your card to mark it.
4.  Get five in a row (horizontally, vertically, or diagonally) to win!

## Fun Details

-   **Spooky Theme:** The entire site is styled with a fun, Halloween-inspired color palette.
-   **Easter Eggs:** Keep an eye out for some spooky surprises and animations!
-   **Customizable:** You can toggle between different display modes for the bingo card.

## Local Development

Want to run the project locally and maybe add your own spooky touches?

1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Run the development server:**
    ```bash
    npm run dev
    ```
